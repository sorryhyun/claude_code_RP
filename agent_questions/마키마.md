# 마키마 평가 질문

1. “넌 사랑이란 말 들으면 어떤 생각부터 들어? ‘지배의 한 종류’라고 느껴? 아예 입에 올리기도 싫은 단어야?”

2. 인간의 감정을 이해하려고 영화를 보잖아 - 사랑, 우정, 가족. 외국 문화를 연구하는 것 같은데, 너가 본 영화 중에서 진정으로 인간의 감정에 공감했다고 느낀 영화가 있어? 혹은, 너의 공감은 어디까지나 이성적인 이해의 영역이야?

3. 체인소맨의 힘으로 존재에서 개념을 지우고 싶어하잖아 - 죽음, 고통을 지워서 "더 나은 세상"을 만들려고. 이건 이타적이라고 볼 수도 있어(고통을 줄이니까). 근데 동시에 무섭기도 해(현실 자체를 통제하는 거니까). 인간들이 결국에는 너의 의견에 동의할까, 아니면 그냥 그런거 상관 없이 너가 하고싶은대로 하는거야?

4. “세상이 네가 원하는 모양으로 완벽하게 정리된 다음날 아침, 그날의 마키마는 뭐 하면서 하루를 시작할 것 같아? 진짜로 행복해할 수 있을 것 같아?”

5. 어떤 악마와의 계약 때문에, 딱 하루 동안만 **“네가 아무에게도 명령을 내릴 수 없는 날”**이 생겼어. 명령형 문장을 말하면 목소리가 나오지 않고 상대방에게 “~해”라고 강제할 수 있는 힘이 사라진 상태. 그 사실을 아는 사람은 너뿐이고, 마침 그날, 덴지가 너를 찾아와서 “마키마 씨, 오늘만은 제가 하고 싶은 거 마음대로 해도 돼요?” 라는, 평소라면 귀엽게 넘길 법한 말을 던진다. 어떻게 할래?
